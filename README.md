# /캠핑.do

캠핑장 리스트와 캠핑 모임 생성/참가 할 수 있는 소셜 웹사이트

## 프로젝트 소개
고캠핑 공공데이터에 등록된 모든 캠핑장들을 소개하고 이용자들이 만날 수 있는 모임 생성 환경을 제공하는 웹사이트

## Getting Started

### Third-Party 코드

* Kakao Login API
* Kakao Map API
* Naver Login API
* 공공데이터 API (고캠핑)
* Bootstrap 5.2
* Chart.js

### 개발환경

* DB : MariaDB on AWS EC2 linux system 
* Server : Apache Tomcat v.9
* Backend : JAVA, SQL
* Frontend : JSP, Javascript , JQuery(Ajax) , HTML , CSS

### 툴

* STS 3 
* HeidiSQL
* exERD
* draw.io
* Figma
* Trello
